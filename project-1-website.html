While making my website, I learned a lot about how to plan, design, and build a website on our own. At the start, I had to decide what the site would look like, which colours to use, what each page looked like. I tried a few ideas before choosing the layout that felt the cleanest and easiest to use. One challenge was getting the site to look good on different screen sizes,I was able to fix that, using css and html and responsive web design. I also had small problems like images not lining up or the boxes not being in the correct place, but I fixed them by testing them and running them I built the site using html and css, and I used github to save my work and keep track of changes. This project helped me understand how these tools work, and how to use them. I learned how important test things, instead of all at the end. In the future, I want to improve the design even more, use javascript, and make the site feel more professional. Overall, this project helped learn more skills on coding and feel more confident building websites.
