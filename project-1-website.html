While making my website, I learned a lot about how to plan, design, and build a real project from scratch. At the start, I had to decide what the site would look like, which colours to use, and how to organise each page. I tried a few ideas before choosing the layout that felt the cleanest and easiest to use. One challenge was getting the site to look good on different screen sizes, but using responsive design and CSS helped me fix that. I also had small problems like images not lining up or text moving around, and I solved them by testing often and adjusting my code. I built the site using HTML and CSS in Visual Studio Code, and I used Git and GitHub to save my work and keep track of changes. This project helped me understand how these tools work together in real web development. I learned how important it is to test things step-by-step instead of trying to fix everything at the end. In the future, I want to improve the design even more, add better animations, and make the site feel more professional. Overall, this project helped me grow my skills and feel more confident building websites.
